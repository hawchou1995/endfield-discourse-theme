## Installation

### Via Admin Panel

1. Go to **Admin > Customize > Themes**
2. Click **Install Theme**
3. Enter the theme URL: `https://github.com/hawchou1995/endfield-discourse-theme.git`
4. Click **Install**
5. Select the theme and choose "Endfield Dark" or "Endfield Light" color scheme
